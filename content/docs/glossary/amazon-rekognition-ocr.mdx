Amazon Rekognition OCR (Text Detection) is part of AWS Rekognition service that uses computer vision and machine learning to detect and extract text from images and videos. It can identify text in various scenarios including street signs, product labels, and documents.

## Key Features

- **Text Detection**: Detects text in images and videos
- **Multi-language Support**: Recognizes text in multiple languages
- **Real-time Processing**: Can process images and video streams in real-time
- **Scene Text Detection**: Detects text in natural scenes, not just documents
- **Bounding Box Information**: Provides location information for detected text
- **Confidence Scores**: Returns confidence levels for detected text

## Use Cases

- Street sign and license plate recognition
- Product label reading
- Social media content moderation
- Video transcription
- Image search and indexing
- Accessibility features for images
