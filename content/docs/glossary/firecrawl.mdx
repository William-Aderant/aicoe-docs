FireCrawl is a web scraping and crawling service that converts websites into clean, structured markdown or structured data. It handles JavaScript rendering, pagination, and complex website structures automatically.

## Key Features

- **Automatic Crawling**: Crawl entire websites or specific pages
- **JavaScript Rendering**: Handles dynamic content and single-page applications
- **Clean Output**: Converts HTML to markdown or structured JSON
- **API-First**: Simple REST API for integration
- **Rate Limiting**: Built-in respect for robots.txt and rate limits

## Use Cases

- Content aggregation
- Data extraction for research
- Website monitoring and change detection
- Building knowledge bases from web content
- Competitive analysis
