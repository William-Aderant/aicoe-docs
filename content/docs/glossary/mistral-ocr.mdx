Mistral OCR, also known as Voxtral, is Mistral AI's open-weight speech recognition model designed to compete with solutions like OpenAI Whisper. It offers advanced speech-to-text capabilities with competitive performance and cost advantages.

## Key Features

- **Open-Weight Model**: Available for local deployment or via API
- **High Performance**: Outperforms Whisper large-v3 in various benchmarks
- **Multilingual Support**: Strong capabilities across multiple languages
- **Enterprise Features**: Supports private deployment, domain-specific fine-tuning, speaker identification, and emotion detection
- **Large Context**: 32K token context window for processing longer audio segments

## Use Cases

- Speech-to-text transcription
- Multilingual audio transcription
- Speaker identification and diarization
- Emotion detection in speech
- Audio understanding and question answering
